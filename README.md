Notes (`ipm-notes`)

A simple note-taking module with bulleted list display.

**Features:**
- Type notes and press Enter to add (Shift+Enter for new lines)
- Right-click or long-press to edit notes
- Scrollable content

**Config:** No configuration needed

```json
{
  "name": "ipm-notes",
  "width": 1,
  "height": 1,
  "x": 0,
  "y": 0,
  "config": {}
}
```
